For property set operations, this will be 0. For collection operations, this reflects the number of items added, removed, or replaced.

Index

public int Index { get; }

Property Value

int

Remarks

For example, during an Add/Remove operation in a list, this represents the index where the change occurred. Returns -1 if not applicable (e.g., property set).
